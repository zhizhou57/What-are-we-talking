## 🚀 快速开始

### 第一步：获取聊天数据
- **QQ用户**：使用[QQ History Backup](https://github.com/Yiyiyimu/QQ-History-Backup) → 保存为`qq_messages.csv`  
- **微信用户**：使用[WeChatMsg](https://github.com/LC044/WeChatMsg) → 保存为`wechat_messages.csv`

### 第二步：运行魔法！
```bash
python analysis.py
```
